About

The Design

At the core of OneReminder is the tried and true tickler file system.

A system that has been used by large organisations to track important tasks since the pre-digital age.

With OneReminder you no longer need to keep large folders of notes, reminders and lists. Instead you can use a simplfied system with notifications and delightful gestures.

Turning bulky folders into two simple pieces: A daily list and a calendar drawer.

In short, OneReminder is the tickler file system redefined for iOS devices.
